Datu Struktūras

Tīmekļa lietotne Datu Struktūru apguvei

Prasības

  • Git
  • Node.js (ar NPM)
  • Bower: npm install bower -g
  • Ember CLI
  • PhantomJS

Uzstādīšana

  • npm install
  • bower install

Palaišana

  • ember server
  • Skaties lietojumu http://localhost:4200.

Testi

  • ember test
  • ember test --server

Kompilēšana

  • ember build (izstrāde)
  • ember build --environment production (produkcija)

Atjaunināšana

Lai atjaunotu Foundation un citas bibliotēkas izpildi

bower update
